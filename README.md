# 중세 카드 경영 (Medieval Card Management)

브라우저에서 바로 실행되는 순수 바닐라 JS 카드 경영 게임.  
빌드 도구 없음, 의존성 없음 — 파일 열기만 하면 바로 플레이 가능.

---

## 빠른 시작

```bash
git clone <repo-url>
cd ai_coding_games
# index.html을 브라우저로 열면 끝 (로컬 서버 불필요)
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

Live Server (VS Code 확장) 사용 시: `index.html` 우클릭 → Open with Live Server

## 게임 개요

카드를 드래그&드롭하여 상호작용시키는 중세 경영 게임.

- **초기 상태**: 사람 카드 3개 + 농지 1개로 시작
- **목표**: 위협(늑대·곰)을 막으며 자원을 늘리고 성벽을 구축

### 카드 종류

| 카테고리 | 카드 |
|---------|------|
| 인류 | 사람, 전사 |
| 건설 | 농지, 망한농지, 영지, 배, 성벽 |
| 자연 (우호) | 바위, 나무, 토끼, 사슴 |
| 자연 (위협) | 늑대, 곰 |
| 재화 | 식량, 돌, 목재, 벽돌 |
| 무기 | 창 |

### 주요 상호작용

```
사람 → 농지          사람이 농지 일함 → 식량 생산 (15초)
사람 → 바위/나무     자원 채취 → 돌 2개 / 목재 2개 (60초, 채취 후 소멸)
사람 + 사람          번식 → 사람 추가 (식량 소모)
창 + 사람            전사로 변환
전사 → 토끼/사슴     사냥 → 식량
전사 → 늑대/곰       전투 (사망 확률 15%/30%)
사람 → 배            탐험 → 랜덤 재화 5개 (60초)
```

### 조합 레시피

```
돌 + 돌       → 벽돌
벽돌 + 벽돌   → 성벽 (HP: 5)
벽돌 + 성벽   → 성벽 HP +3 (수리)
목재 + 돌     → 창
목재 + 목재   → 배
```

### 자연 스폰 주기 (게임 시간 기준, 초)

| 카드 | 첫 등장 | 이후 주기 | 최대 수 |
|-----|--------|---------|-------|
| 바위 | 30초 | 40~80초 | 3 |
| 나무 | 30초 | 40~80초 | 3 |
| 토끼 | 45초 | 50~100초 | 3 |
| 사슴 | 60초 | 70~140초 | 2 |
| 늑대 | 100초 | 100~200초 | 2 |
| 곰 | 180초 | 150~300초 | 1 |

위협 카드(늑대·곰)를 처치하지 않으면 타이머 만료 시 카드 파괴.  
성벽이 있으면 피해를 대신 흡수 (HP 소진 후 파괴).

## 코드 구조 (game.js)

```
CARD_DEFS       카드 정의 (이름, 아이콘, 타이머, 작업 설정)
COMBINATIONS    조합 레시피 배열
SPAWN_CONFIG    자연 스폰 타이밍 설정
─────────────────────────────────────────
init()              게임 초기화, 첫 카드 배치
gameLoop()          requestAnimationFrame 루프, dt 계산
updateCards(dt)     타이머 틱, 만료 처리, 작업 완료 처리
─────────────────────────────────────────
createCard()        카드 생성 + DOM 추가
removeCard()        카드 제거 (dying 애니메이션)
transformCard()     카드 타입 변환 (DOM 교체)
buildCardDOM()      카드 HTML 요소 생성
─────────────────────────────────────────
canInteract()       두 카드가 상호작용 가능한지 판단
executeInteraction() 실제 상호작용 실행
─────────────────────────────────────────
setupEventListeners()  마우스/터치 이벤트 등록
onMouseDown/Move/Up    드래그 처리
findCardAt()           좌표로 카드 탐색
findOverlapping()      겹치는 카드 탐색
```

## 개발 팁

- 수정 후 브라우저 새로고침만 하면 반영됨 (빌드 없음)
- `gameSpeed` 전역 변수로 속도 배율 관리
- 모든 타이머는 `dt * gameSpeed` 단위로 감소 (초 단위)
- 카드 상태는 `cards[]` 배열에서 관리, DOM은 `card.element`로 접근
